Detailed Description

A deadly laser that brings destruction to anything in it's wake.

A "one hit kill" laser with a static base. The length of a laser is "infinite" (big enough to cover whole screen) but it will be blocked by a first object the laser ray collides with.

Laser can be static (not move at all), or have a defined base length in which case it will slide on it.

Laser has it's own color, so if it is picked as active, the laser will dissapear leaving only the base. The base is always black.
